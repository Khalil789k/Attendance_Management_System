# Contributing

Want to help out? Great! Here's how:

1.  **Found a bug?** Open an issue on GitHub and let us know what's wrong.
2.  **Have a fix?** Fork the repo, make your changes, and send a Pull Request.
3.  **New Feature?** Describe your idea in an issue first so we can chat about it.

Keep the code clean and test it before you send it over. By contributing, you agree that your code will be under the same MIT license as the project.

Thanks for helping!
